# 🏥 Clinic Management System
# نظام إدارة العيادة الطبية

Modern medical clinic management system built with React, Node.js, TypeScript, and PostgreSQL.

نظام حديث وشامل لإدارة العيادات الطبية باستخدام أحدث التقنيات.

## ✨ Features | المميزات

- 👥 **Patient Management** - إدارة المرضى
- 📅 **Appointment Scheduling** - حجز المواعيد
- 👔 **Staff Management** - إدارة الموظفين
- 💰 **Billing System** - نظام الفواتير
- 📋 **Medical Records** - السجلات الطبية
- 📊 **Reports & Analytics** - التقارير والإحصائيات
- 🔐 **Authentication** - المصادقة والأمان
- 🌍 **Multi-Language** - دعم اللغة العربية والإنجليزية

## 🛠️ Technology Stack

### Backend
- Node.js + Express
- TypeScript
- PostgreSQL
- JWT Authentication
- Docker

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router

## 📦 Prerequisites

- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 14+
- npm or yarn

## 🚀 Quick Start

### Using Docker (Recommended)
```bash
git clone https://github.com/bouaicidz/clinic-management.git
cd clinic-management
docker-compose up -d
```

Access:
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
- Database: localhost:5432

### Manual Setup

#### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

#### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```
